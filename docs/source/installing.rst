Installing harmonize_wq
=======================

Install harmonize_wq using pip:
*******************************

.. code-block:: python3
   
    pip install harmonize-wq



Install the latest development version of harmonize_wq using pip:
*****************************************************************

.. code-block:: python3
   
    pip install git+https://github.com/USEPA/harmonize-wq.git
